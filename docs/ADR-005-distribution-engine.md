# ADR-005: محرك التوزيع متعدد الأنماط

**الحالة**: معتمد | **التاريخ**: 2024  
**السياق**: الجمعية تحتاج توزيع المساعدات بطرق مختلفة (حسب العمر، المرحلة الدراسية، عدد الأطفال).

**القرار**: محرك توزيع بثلاثة أنماط مع شرائح قابلة للتخصيص.

**الأنماط**:
1. **age** — شرائح عمرية (من - إلى) مع مبلغ لكل شريحة
2. **school_stage** — شرائح حسب المرحلة التعليمية
3. **children_count** — حسب عدد الأطفال في الأسرة

**الإعدادات**:
- `is_auto_calculate` true → يحسب لكل طفل
- `is_auto_calculate` false → مبلغ ثابت لكل أسرة
- `commission_rules` → عمولة التنفيذ (اختياري)

**التخزين**: brackets مخزنة كـ JSONB في `campaigns` (age_brackets, stage_brackets, children_brackets, commission_rules)

**اعتبارات لاحقة**:
- دعم التوزيع المختلط (Mixed Mode)
- دعم التوزيع حسب المنطقة الجغرافية
