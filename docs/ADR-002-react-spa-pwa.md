# ADR-002: React SPA + PWA

**الحالة**: معتمد | **التاريخ**: 2024  
**السياق**: تطبيق داخلي لجمعية خيرية، يحتاج للعمل في مناطق ضعيفة الاتصال.

**الخيارات**: Next.js (SSR), React SPA + Vite, Vue.js.

**القرار**: React 19 + Vite 8 + vite-plugin-pwa.

**الإيجابيات**:
- PWA للتثبيت على الموبايل والعمل دون اتصال
- Service Worker للتخزين المؤقت
- سرعة تطوير عالية مع Vite HMR

**السلبيات**:
- لا SSR (غير مطلوب لتطبيق داخلي)
- SEO غير مهم (يحتاجه المسجلون فقط)

**اعتبارات لاحقة**:
- `offlineStore.ts` مع Zustand persist يوفر التخزين المحلي
- `syncQueue` لمزامنة الإجراءات عند العودة للاتصال
